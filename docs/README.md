# Siavash Torkamannejad — Developer Portfolio

A responsive personal portfolio website presenting my background, technical skills, professional experience, and selected web-development projects.

## Sections

- Hero introduction
- About
- Technical skills and proficiency levels
- Work experience
- Projects and write-ups
- Contact links

## Technology

- HTML5
- CSS3
- Vanilla JavaScript
- Google Fonts

## Highlights

- Responsive single-page layout
- Clear project and skills presentation
- Accessible semantic structure
- Lightweight frontend with no framework or build step
- Direct links for professional contact

## Run Locally

```bash
git clone https://github.com/siavashtnejad-ux/Portfoliooo.git
cd Portfoliooo
python3 -m http.server 8000
```

Then open `http://localhost:8000`.

You can also open `index.html` directly in a modern browser.

## Customization

All content and styling are contained in `index.html`. Update the biography, skills, projects, experience, and contact links there.

## Deployment

This static website can be hosted on GitHub Pages, Netlify, Cloudflare Pages, Vercel, or any standard web server.
